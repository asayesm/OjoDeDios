# OjoDeDios
## Terminos de Privacidad de Aplicación DEMO
```
El aplicativo DEMO Ojo de Dios es un aplicativo que extrae la información de ubicación del usuario para mostrar su geolocalización en el mapa de Google, dichos datos son usados solamente para la simulación de que se registran sucesos en lugares donde se encuentre los usuarios.

Dicha información no se almacena en ningún lado solamente es para la mejora de la interacción con el usuario y el aplicativo, al cerrar el aplicativo y volver a entrar, la información generada no se vuelve a visualizar puesto que solo se muestra durante el uso y está es cargada temporalmente en la programación.

La información de pruebas que se muestra en el aplicativo es información ficticia hecha para poder mostrar posibles datos en la interfaz.


```
